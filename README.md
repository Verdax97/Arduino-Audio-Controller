# Arduino Audio Controller
## Requirements
* Arduino:
	* Libraries:
		* ArduinoJson
		* Adafruit_GFX
		* Adafruit_SSD1306
	* Components:
		* 2 potentiometers
		* lcd oled screen
		* 1 button
		* 1 resistor
		* lots of cables
* Python3:
	* Libraries:
		* argparse
		* time
		* serial
		* json
		* pycaw
